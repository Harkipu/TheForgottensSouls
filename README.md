## About
The Forgotten, a normal soldier in medieval fantasy world which is a ‘nobody’ that no one know him.<br>
While he was exploring in a forest at night, he accidentally fell into an underground.<br>
The underground is surrounded with lots of monsters around.<br>

---

## Controls

Mouse left click to confirm

Attack to do Basic attack
Skills to do Magic attack
Guard to block enemy attack

---

##  Scripts

|  Scripts | Description |
| --- | --- |
| `AudioSettingManager.cs` | Script that manages the Audio modifier in the game settings.|
| `BattleHUD.cs` | Big script to control nearly all the UI that appears in the game including buttons.|
| `BattleManager.cs`| Big script that control nearly all the interaction and execution of action like damage, attack, wave, stage, enemy spawn, damage overtime, and etc.|
| `EnemyStage.cs`| Script to maintain and get data of enemy spawn and their spawn location that would be use by the other script.|
| `GameOverScript.cs`| Script to show gameover and what happen after gameover.|
| `LoadSceneScript.cs`| Tiny script to change scene (main menu scene to battle scene).|
| `MenuUIManager.cs`| Script that manage the UI in mainmenu.|
| `SelectSkills.cs`| Script that manage the skills that player choose and effect that certain skill give.|
| `Character`| Folder that hold script mostly related to character data.|
| `Enemy`| Folder similar like Character folder.|
| `ScriptableObj`| Folder that contain all raw data of things like player hp, various enemy attack, status condition, skills, etc.|
